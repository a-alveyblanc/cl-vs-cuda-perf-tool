Quick analysis tool for dumping PTX/SASS and other compiler artifacts from
OpenCL and CUDA kernels generated by loopy.

Terminal view for quickly viewing things like spills, # SASS instructions, #
FFMAS, mem ops, etc.

Requires an experimental loopy branch: https://github.com/a-alveyblanc/loopy/tree/compute-with-precise-dependencies
Requires an experimental namedisl branch: https://github.com/a-alveyblanc/namedisl/tree/initial-functionality
